# Music Genre Classification

🎵 Music Genre Classification



This project is a Machine Learning model that classifies music into different genres based on audio features.



\---

📌 Project Description

The system takes audio files as input, extracts important features using \*\*librosa\*\*, and then predicts the genre using a trained \*\*Machine Learning model (RandomForest)\*\*.



\---



&#x20;🚀 Features

\- Audio feature extraction using librosa

\- Machine Learning model training

\- Data scaling using StandardScaler

\- Model saved using pickle (`model.pkl`)

\- Scaler saved using pickle (`scaler.pkl`)



\---



&#x20;📂 Files in this Project

\- `main.py` → Main Python script for training and prediction

\- `model.pkl` → Trained ML model

\- `scaler.pkl` → Feature scaler

\- `temp.wav` → Sample audio file

\- `.code-workspace` → VS Code workspace file



\---
